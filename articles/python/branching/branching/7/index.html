Ветвление

Ветвление нужны программе когда в ее алгоритме есть действия которые должны быть выполнены, только в зависимости от определенных условии.

Практика

A1. Курьерская служба

Входные данные

  • W = Вес посылки
  • A = Расстояние
  • Является ли повторной?

    • Минимальная сумма за предоставление услуги 200 рублей. На минимальную сумму доставки происходит наценка P.

    P = W * 5 * A

    Если курьеру приходится делать повторную доставку тогда к итоговой сумме прибавляется дополнительная плата RP.

    RP = 100 + W * 3 * A

    Программа должна подчитать итоговую сумму заказа.

    A2. Библиотечный реестр

    Входные данные

  • Количество дней на которое библиотека предоставляет книги
  • Количество книг которое взял читатель
  • Количество дней через которые книги были возвращены

    • Программа должна определить допустил ли читатель просрочку. Если просрочка тогда должно быть выдано предупреждение и выдан штраф. В ином случая предложение взять еще книг. Штраф считается по формуле

    20 * Количество книг * Количество дней

    A3. Оптовая продажа

    Входные данные

  • Количество единиц для закупки
  • Стоимость за единицу

    • Рассчитайте итоговую сумму закупки. Если закупается более 30 единиц, то на каждую единицу делается уценка в 10%. Если более 70 тогда 15%, более 120 20%, более 150 25%
