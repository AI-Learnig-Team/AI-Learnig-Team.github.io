Ветвление

Ветвление нужны программе когда в ее алгоритме есть действия которые должны быть выполнены, только в зависимости от определенных условии.

Сложные логические выражения

В некоторых случаях одного условия бывает недостаточно. Соедения нескольких выражении осуществляются специальными операторами И и ИЛИ.

a < b and b > c  # (True когда a<b = True и b>c = True)

a < b or a < c  # (True когда a<b = True или a<c = True)

Так же существует оператор отрицания НЕ.

not a<b  # (True когда a<b False)

a = 2
b = 3
c = 4
