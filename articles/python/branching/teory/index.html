Ветвление

Ветвление нужны программе когда в ее алгоритме есть действия которые должны быть выполнены, только в зависимости от определенных условии.

1. Реализация

Как и в большинстве языков программирования в Python начало такого ветвления обозначается командой if, описанием условия и двоеточием на конце.

if условие:

Что бы обозначить Python содержимое и конец ветвления необходимо код ветвления писать с отступом от уровня основного кода.

if условие:
    код ветвления

основной код

2. Логические выражения

Для описания условия используются логические операторы.

Оператор Значение
> Больше
< Меньше
>= Больше или равно
<= Меньше или равно
== Равно
!= Не равно

if a < b: #Если a меньше b

Описание условия «a < b» называется логическим выражением. Логическое выражение может иметь только два значения: Истина или Ложь. В Python обозначается как True и False. if срабатывает только если выражения в нем принимает значения True (Правда).

a = 2
b = 3

if a > b:
    код ветвления # Этот код не выполнится так как выражение равно False

if a < b:
    код ветвления # Этот код выполнится!

3. Две ветви

Все выше описанное реализует скорее ответвление от основного алгоритма. Для реализации полноценного ветвления так же используется команда else.В отличие от if ей не требуется логического выражения, ей обозначается начало ветвления которое срабатывает если выражение в if было False. Тем самым его легко можно заменить в голове на слово иначе.

if a > b:
    код первой ветки
else:
    код второй ветки

4. Сложные логические выражения

В некоторых случаях одного условия бывает недостаточно. Соедения нескольких выражении осуществляются специальными операторами and(И) и or(ИЛИ).

if a > b and a > c:# Сработает только если оба выражения истинны

if a > b or a > c:# Не сработает только если оба выражения ложны

При необходимости инвертировать значения выражения используется оператор not(Не).

if not a > b: # Сработает только если выражения ложно

5. Множество ветвей

В случаях когда требуется более двух ветвей, используется команда elif. Данная команда идет после if, но до else и так же как и if требует логического выражения.

if a > b and a > c:
    код первой ветки
elif b > a and b > c:
    код второй ветки
elif c > a and c > b:
    код третьи ветки
else:
    код четвертой ветки

6. *Еще немного о бинарной логики

Логические выражения, как и математические операции можно сохранять в переменную. Тип такой переменой в Python называется Boolean. Это тип, как и выражения, может иметь только два значения (True or False, 0 or 1 в числовом представлении).

a = b > c #При b=2,с=1 a=True

Так же выражения могут просто True или False. Что означает что выражение всегда Истинно или Ложно.

if True: #Такое ветвление будет исполниться всегда

if False: #Такое ветвление будет исполниться никогда

7. *Короткие условия

Если два ветвления состоят из простых однострочных операции, то имеет смысл использовать короткий формат записи.

Вместо:

if a > b:
    c = 15
else:
    c = 25

Можно писать так:

c = 15 if a > b else 25
