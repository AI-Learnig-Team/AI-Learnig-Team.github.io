Ветвление

Ветвление нужны программе когда в ее алгоритме есть действия которые должны быть выполнены, только в зависимости от определенных условии.

5. Множество ветвей

В случаях когда требуется более двух ветвей, используется команда elif. Данная команда идет после if, но до else и так же как и if требует логического выражения.

if a > b and a > c:
    код первой ветки
elif b > a and b > c:
    код второй ветки
elif c > a and c > b:
    код третьи ветки
else:
    код четвертой ветки
